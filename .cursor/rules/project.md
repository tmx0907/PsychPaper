# 프로젝트 개요

**PsychPaper** — 심리학 논문 해설 & 지식 탐색 플랫폼

## 핵심 컨셉

- **논문 해설 블로그**: 학술 논문을 일반인이 읽기 쉬운 글로 해설해 공유
- **PARS 탐색**: 주제/현상/응용/연구방법별로 분류한 카드 그리드로 논문 탐색
- **Deep Dive 아티클**: 스크롤 기반, 데이터 시각화가 포함된 몰입형 HTML 아티클
- **SNS 유입 루프**: 요약 카드 공유 → 링크 클릭 → 웹 유입 → 탐색·댓글·재공유

## 사용자

| 구분 | 역할 |
|------|------|
| **1차 (Admin)** | 작성자 1인. 아티클 작성·편집·발행, PARS 태그·카테고리 관리, 댓글 승인/삭제 |
| **2차 (Reader)** | 독자 다수. 글 탐색·읽기, 댓글(닉네임·비밀번호 선택), SNS 공유. MVP에서는 로그인 불필요 |

## 사용 환경

- PC 브라우저 우선 (Deep Dive 시각 경험)
- 모바일 반응형 필수 (SNS 유입 대응)

## PARS (분류 프레임워크)

- **P**henomenon: 어떤 심리 현상인가 (수면·정서·인지·관계 등)
- **A**pplication: 실생활 적용 (학습·직장·건강·육아 등)
- **R**esearch Type: 연구 유형 (메타분석, RCT, 종단 등)
- **S**cale: 규모감 (소규모 ~ 메가, 참가자 수)

각 아티클은 4축에서 각각 하나 이상 태그 보유.

## MVP에서 제외

독자 로그인, 좋아요/북마크, 뉴스레터, 옵시디언 그래프 뷰, 다국어, AI 요약 자동생성, OG 이미지 자동생성, 조회수 중복 방지, 전문 검색 고도화.

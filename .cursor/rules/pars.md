# PARS 카테고리 체계

플랫폼 논문 분류 프레임워크. 독자가 "어떤 관점으로 찾고 싶은가"에 따라 4축.

## P — Phenomenon (현상)

예: 수면&각성, 정서&기분, 인지&학습, 관계&사회, 스트레스&대처, 동기&행동, 발달&노화, 성격&개인차

## A — Application (응용)

예: 학습&교육, 직장&생산성, 건강&웰빙, 관계&소통, 육아&발달, 자기이해

## R — Research Type (연구 유형)

메타분석, 체계적 문헌고찰, RCT, 종단 연구, 횡단 연구, 질적 연구

## S — Scale (규모감)

| 라벨 | 기준 |
|------|------|
| 소규모 | 참가자 100명 미만 |
| 중규모 | 100~1,000명 |
| 대규모 | 1,000~100,000명 |
| 메가 | 100,000명 이상 |

각 아티클은 4축에서 각각 하나 이상의 태그(카테고리)를 가짐. 자유 태그(tags)는 별도.

## UI

- 홈: PARS 탭(현상/응용/연구유형/규모) 전환 시 해당 축 기준 카드 그리드
- 아티클 카드·상세: PARS 뱃지 클릭 시 해당 태그 목록으로 이동 (옵시디언 백링크 느낌)
